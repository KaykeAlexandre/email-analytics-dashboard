# Email Analytics Dashboard

## Visão Geral
Projeto de análise de campanhas de e-mail marketing desenvolvido para monitorar métricas de desempenho, engajamento e comportamento dos usuários através de dashboards e consultas SQL.

## Ferramentas Utilizadas
- SQL
- Google Looker Studio
- Google Sheets
- Visualização de Dados
